\subsection{Detailed Description}
Vérifie si un nombre est premier. 

Cette fonction utilise une méthode de vérification simple pour déterminer si un nombre est premier. Elle vérifie d\textquotesingle{}abord si le nombre est inférieur ou égal à 1 (non premier), égal à 2 (premier), ou pair (non premier). Pour les autres nombres, elle vérifie la divisibilité par des nombres impairs jusqu\textquotesingle{}à la racine carrée du nombre.
